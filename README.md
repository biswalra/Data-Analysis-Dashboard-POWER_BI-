# HR (Analysis Dashboard using PowerBi)
## Project Objective
To analyze employee data and visualize key HR metrics like attrition, demographics, and job satisfaction to support better workforce decision-making. 📊

# DataSet Used
<a href="https://github.com/biswalra/Data-Analysis-Dashboard-POWER_BI-/blob/main/HR%20DATA_Excel%20(1).xlsx"> HR Data </a>

## KPI-Based Questions

- What is the total number of employees in the organization?
- What is the overall employee attrition count and attrition rate?
- How many employees are currently active?
- Which department has the highest attrition?
- What is the average age of employees?
- How does attrition vary across different age groups?
- What is the job satisfaction level across different job roles?
- Which educational field has the highest attrition?
